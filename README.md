# Taxi Fare Prediction

A Machine Learning web application that predicts taxi fares based on trip distance, location, and travel details.

## Features

- Fare prediction using Machine Learning
- Data preprocessing and feature engineering
- Regression model training
- Interactive user interface
- Real-time fare estimation

## Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit

## Dataset

Uber Taxi Fare Dataset (50,000 trips)

## Machine Learning Workflow

1. Data Cleaning
2. Feature Engineering
3. Model Training
4. Model Evaluation
5. Prediction Interface

## Future Improvements

- GPS integration
- Route optimization
- Multiple ML model comparison
- Live deployment
